<p><a href="metalsmith.io.html">metalsmith</a> plugin to detect mime types from file contents</p>
<p>take a look at the <a href="https://leviwheatcroft.github.io/metalsmith-mime-type/lib/index.js.html">annotated source</a></p>
<div class="pilwrap" id="install">
  <h2>
    <a href="#install" name="install" class="pilcrow"></a>
install
  </h2>
</div>
<p><code>npm i --save metalsmith-mime-type</code></p>
<div class="pilwrap" id="usage">
  <h2>
    <a href="#usage" name="usage" class="pilcrow"></a>
usage
  </h2>
</div>
<pre><code>mimeType = require('metalsmith-mime-type')

metalsmith(__dirname)
.use(mimeType('**/*'))
.build(callback)
</code></pre>
<div class="pilwrap" id="options">
  <h2>
    <a href="#options" name="options" class="pilcrow"></a>
options
  </h2>
</div>
<p>A single option, <code>src</code> is used as a minimatch mask to determine what files
need to to have mimetypes assigned. can be passed in as an object property,
or as a string. See usage example.</p>
